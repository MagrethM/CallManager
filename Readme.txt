In order to run any of the file, go to command line interface and navigate to the folder where they are stored, also remember to set CLASSPATH to where the .class files are stored.

To compile:javac file_name.java   

Note: SingleServer,SingleServerD,SingleServerK have two classes with the same name, so when compiling these files, there class files should be stored differently
To compile CallManager, compile the files in the following order Timer.java-->SimulItem.java-->ListException.java-->RandomGen.java-->List.java-->CallManager.java

To run CallManager file:
java CallManager
